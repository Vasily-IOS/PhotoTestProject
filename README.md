# PhotoTestProject - приложение для поиска фотографий с помощью открытого API сайта [Unsplash](https://unsplash.com/)

<img width="1440" alt="Снимок экрана 2022-02-07 в 00 50 06" src="https://user-images.githubusercontent.com/84737606/152702861-53f8c306-1a18-4072-bf08-f979956e2c03.png">

Главный экран показывает случайные фотографии, но вы также можете 
искать фотографии на любую тему:

<img width="1440" alt="Снимок экрана 2022-02-07 в 00 50 32" src="https://user-images.githubusercontent.com/84737606/152702954-d600244d-e3e5-4ec3-b079-e01281a22f60.png">

К примеру, найти фото медведей:

<img width="1440" alt="Снимок экрана 2022-02-07 в 00 50 42" src="https://user-images.githubusercontent.com/84737606/152702967-94c06133-e8f8-4e8c-9986-351daccc3a61.png">
